5 eventos importante en la historia de el internet

  • Arpanet
  • Protocolo TCP/IP
  • Nacimiento del WWW
  • Comercializacion del Internet
  • Creacion de redes sociales

1. Creacion de ARPANET(1969)

Se establece ARPANET, la primera red de computadoras, desarrollada por el Departamento de Defensa de EE.UU

Impacto
  • Introduccion del concepto de comunicacion en red.
  • Base para el desarrollo de internet moderno.

2. Creacion del Protocolo TCP/IP

Se implementa el protocolo TCP/IP, permitiendo la cominicacion estandarizada entre redes.

Impacto
  • Se establece la estructura tecnica fudamental del internet
  • Facilita la interoperatividad entre diferentes redes.

3. Nacimiento de la World Wide web (WWW) (1991)

Tim Berners-Lee lanza la primera pagina web y desarrolla el HTML, HTTP y URLs.

Impacto
  • Democratizacion del acceso a la informacion
  • Crecimiento exponencial de sitios web y la era digital.

4. Expancio del Internet Comercial (1995)

Se permite el uso comercial del internet y surgen empresas como Amazon y Ebay.

Impacto
  • Internet deja de ser exclusivo para instituciones academicas y militares
  • Nace el comercion electronico y la economia digital.

5. Creacion de las Redes Sociales(2004-2006)

Facebook (2004) y Twitter (2006) revolucionan la comunicacion en linea.

Impacto
  • Cambio en la forma de socializar y difundir informacion.
  • Mayor interaccion globa y aparicion de fenomenos como la viralizacion.

¿Para que sirven las etiquetas anclas?

Las etiquetas anclas en HTML sirven para crear enlaces que permiten navegar entre páginas, secciones dentro de una misma página o descargar archivos.
